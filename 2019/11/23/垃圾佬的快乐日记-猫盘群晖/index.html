垃圾佬的快乐日记-猫盘群晖

发表于 2019-11-23

猫盘其实已经上车好久了，最近论坛里出了单群晖固件使得猫盘上车价格从最初的40包邮上升到了120，今天特地记录一下这个刷机的全过程

工具：tftp.exe CH340Gttl串口线 ssh工具（putty、mobaxterm等）winhex

固件： 花王网络制作 感谢分享

过程：

1.刷入uboot flash-image.bin

由于官方原版uboot不支持usb/tftp启动方式，所以第一步刷入花王网络修改的uboot（个人理解类似于安卓刷机的recovery）
刷入方式有多种，如果使用过矿渣论坛里发布的猫盘刷onespace工具的话可以直接在onespace的后台命令行里刷入，本文中介绍的是使用ttl刷入，首先是ttl的接法，来自libgcc,要注意ch340g上的tx接猫盘的rx，ch340g上的rx接猫盘的tx，vcc不要接

打开tftp.exe工具，将固件放入tftp目录
插入ttl线之后打开mobaxterm新建serial连接，端口选择ch340g的com口波特率115200，插上猫盘电源后能看见屏幕上输出猫盘的启动日志，连续按键盘任意键打断猫盘启动，进入猫盘的uboot命令模式
命令行输入

重启后再次按键盘打断猫盘启动即可进入修改版的uboot命令行

2.修改群晖8M固件.bin

使用winhex或者UEditor工具修改固件中的MAC和SN以及校验码，搜索001132定位到MAC的位置，替换，校验码计算方法百度

3.将修改后的固件写入spi

打开tftp工具，将修改后的群晖固件.bin放入tftp目录
mobaxterm serial连接中命令行输入

4.装入硬盘，浏览器访问猫盘ip:5000开始安装

后续

1.安装transmission

套件中心-设置-常规-勾选Synology Inc.和信任的发行者
套件中心-设置-套件来源-新增：http://packages.synocommunity.com/
保存后，在 社群 中找到transmission安装，此时的transmission为原版界面，接下来安装ronggang的汉化界面
点击 控制面板-终端机和SNMP-启动SSH功能
mobaxterm新建连接，ip为当前群晖的ip，用户名admin密码为群晖的admin密码
输入命令

提示安装完成后，重新打开transmission（可能需要Ctrl+F5强制刷新浏览器页面）

2.安装entware（qbittorrent，aria2）

安装entware与安装transmission类似，添加源https://www.cphub.net，然后在社群中找到Easy Bootstrap Installer安装，安装时选择entware，安装完成后，再次进入mobaxterm群晖的命令行中

安装完成后，qb可以通过群晖ip:9080打开管理页面，aria2可以通过pandownload、Transdroid（安卓）等客户端管理，也可以使用群晖的创建建站功能创建一个ariang页面

3.设置硬盘休眠

进入mobaxterm群晖的命令行编辑syslog-ng.conf文件

登陆群晖，控制面板-电源设置-硬盘休眠-勾选启动硬盘休眠和启动硬盘休眠日志-重启群晖
